system [function]

FBtoC does not implement the system() function for these argument constants:
_lastCurs [0]
_aplVol [1]
_sysVol [2]
_cpuType [12]
_machType [13]
_aplFlag [14]
_lastCursType [17]

The constants themselves are undefined, and the system() function returns 0 if called with the corresponding numeric value.
